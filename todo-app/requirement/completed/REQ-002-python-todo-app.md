# [REQ-002] Python Todo Application

## Summary
Create a Python todo application that supports creating, listing, completing, and deleting tasks with local persistence.

## User Story
As a user, I want to manage my daily tasks from a simple command interface, so that I can track what is pending and what is done.

## Acceptance Criteria
- [x] User can add a task with non-empty text.
- [x] User can list all tasks with task id, status, and text.
- [x] User can mark a task as completed by id.
- [x] User can delete a task by id.
- [x] Tasks persist across runs in a local JSON file.
- [x] Invalid operations (empty text, unknown id) return clear errors.

## Priority
P1: High
